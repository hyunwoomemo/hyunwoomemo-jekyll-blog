# 리액트 시작하기

## 생성

```console
npm init react-app . 
```

## 시작

```console
npm run start
```

>커맨드요약
>- create-react-app으로 리액트 `프로젝트 생성` <br/>
> `npm init react-app <폴더 이름>` <br/>
> 혹은 폴더를 VS Code로 열고 터미널에서 `npm init react-app .`
>- 개발 모드 실행 <br/>
> `npm run start`
> - 개발 모드 종료<br/>
> Ctrl + C
